1) Icons - top

This template uses the open source icon font ionicons which contains more than 600 resizeable icons. You can use ionicons.com search feature to easily find the icon you want to use. Once you've copied the desired icon's CSS classname, simply add the class 'icon' and the desired icon's classname, such as ion-home to an HTML element.

<span class="icon ion-home"></span>

You can then easily resize and change the color of your icon with the CSS properties 'font-size' and 'color'.

3) Animations - top

This template uses the CSS animation library animate.css completed by the javascript library wow.js to create beautiful animations that reveal as you scroll. To animate an HTML element, select an animation name on animate.css and simply add it as a class of the element. If you want the animation to start only once you scroll past it, simply add the classname 'wow' to your element.

<div class="wow fadeInUp"></div>

For more information about advanced options, please refer to the respective documentation pages of animate.css and wow.js.

4) jQuery scrollTo - top

This template uses the jQuery library scrollTo and the jQuery One Page Navigation Plugin to create smooth scrolling animations to navigate within the page.

In order to set up a navigation bar, you first need to make sure that its markup is as followed:

<ul id='nav'>
   <li class='current'><a href="#section-1">Section 1</a></li>
   <li><a href="#section-2">Section 2</a></li>
   <li><a href="#section-3">Section 3</a></li>
</ul>

<div id='section-1'>This is section 1</div>
<div id='section-2'>This is section 2</div>
<div id='section-3'>This is section 3</div>

Then, simply call the plugin on the navigation with this javascript code:

$(document).ready(function() {
  $('#nav').onePageNav();
});

If you would like a button (#my-button) to bring the user to a specific HTML element (#target-element) with a scroll animation, simply add this javascript code:

$('#my-button').click(function(){
   $.scrollTo( $('#target-element').offset(), 800 );
});

There are many other ways to specify the target of a scrollTo animation that you can discover on the jQuery.ScrollTo Documentation Page

4) Owl Carousel & Nivo Lightbox - top

This template uses the jQuery plugins Owl Carousel and Nivo Lightbox to display responsive and touch friendly carousels. In order to setup an new carousel, you first need to add the following markup :

<div id="owl-example" class="owl-carousel">
   <div> Your Content </div>
   <div> Your Content </div>
   <div> Your Content </div>
</div>

Then, simply call the plugin and the carousel is ready:

$(document).ready(function() {
   $("#owl-example").owlCarousel();
});

If you want your carousel items to appear in a lightbox (pop-up) when clicked, you will need to create a link to the content you want to display in the lightbox (for exemple an image) and then call th Nivo Lightbox plugin on this link, as shown below :

<div id="owl-example" class="owl-carousel">
   <div>
      <a href="image_large.jpg" title="This is an image title">
         <img src="image_thumb.jpg"/>
      </a>
   </div>
   <div>
      <a href="image_large.jpg" title="This is an image title">
         <img src="image_thumb.jpg"/>
      </a>
   </div>
</div>
$(document).ready(function() {
   $('#owl-example a').nivoLightbox();
});

To learn more about advanced settings that you can use to customize your carousel, please refer to the respective documentation pages of Owl Carousel and Nivo Lightbox.

5) Retina Images - top

This template uses the javascript plugin retina.js to serve @2x retina images for HiDPI screens. That means that for every image that is displayed with the HTML img tag, this script will search for the corresponding high resolution image that uses Apple's prescribed high-resolution modifier (@2x).

For example, if you have an image on your page that looks like this:

<img src="/images/my_image.png" >

The script will check your server to see if an alternative image exists at this path:

"/images/my_image@2x.png"

To include retina graphics when displaying an image via the CSS property 'background', add the retina-specific rule at the very end of the style.css file, inside the apropriate media query :

@media (-webkit-min-device-pixel-ratio: 1.1), (min-resolution: 120dppx){
   /* Retina-specific rule(s) here */
}
